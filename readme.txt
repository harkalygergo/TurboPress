Multilingual and multisite compatible WordPress plugin that provides a set of tools and features to enhance the functionality of any website. Make it better, faster and safer. It's free and always will be.

== Description ==

Ezzel a bővítménnyel egy plusz főmenü keletkezik a WordPress admin felületén, ahol számos beállítást és funkciót találsz, amelyek segítenek a weboldalad gyorsabbá, biztonságosabbá és jobbá tételében. A bővítmény ingyenes és mindig az is lesz.

== Installation ==

1. Töltsd le a bővítményt közvetlenül a GitHub-ról vagy a WordPress.org oldaláról
2. Aktiváld a bővítményt a WordPress admin felületén
3. A bővítmény beállításai a WordPress admin felületén található TurboPress menüpont alatt érhetők el
4. A bővítmény használatához nincs szükség további beállításokra, de érdemes átnézni a beállításokat és testreszabni a weboldalad igényei szerint
